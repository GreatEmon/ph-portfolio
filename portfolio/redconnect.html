RedConnect - Blood Donation Application

A MERN stack platform connecting donors, volunteers, and those in critical need of blood.

Project Overview

RedConnect is a user-friendly, full-stack application built using the MERN stack (MongoDB, Express, React, Node.js). The platform is designed to streamline the blood donation process by efficiently connecting donors with recipients.

It features a complex role-based system (Admin, Donor, Volunteer) to manage donation requests, user profiles, blogs, and fundraising through secure Stripe integration. The application's goal is to increase donation efficiency and maintain accurate, accessible records.

Key Implementation Challenges

  • Robust Role Management: Designing and enforcing strict role-based access control (RBAC) across the application (Admin, Donor, Volunteer) using Firebase Authentication and protecting private APIs with JWT.
  • Complex Data Filtering: Implementing efficient database queries for searching donors by multiple parameters (blood group, district, upazila) while maintaining performance.
  • External Integrations: Successfully integrating the Stripe payment gateway for the funding system and handling subsequent transaction management and data tabulation with pagination.
  • Content Management: Integrating and customizing jodit-react as a rich text editor to provide a seamless blogging platform for administrators.

Potential Improvements & Future Plans

  • Geo-Spatial Search: Upgrade donor search to use MongoDB's GeoJSON capabilities to identify and display the nearest available donors based on the recipient's location, improving emergency response times.
  • Real-Time Notifications: Integrate WebSockets (e.g., Socket.io) to enable real-time updates and push notifications for status changes in donation requests.
  • Automated Reporting: Enhance the admin dashboard with scheduled reporting and deeper analytics on donation request density and volunteer activity across different regions.
