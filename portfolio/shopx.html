Project Purpose

ShopX is a comprehensive B2B wholesale marketplace designed to facilitate secure, high-volume transactions between manufacturers, distributors, and institutional buyers. Built on the MERN stack, the platform manages complex inventory logic, multi-category listings, and strict authentication necessary for enterprise commerce.

This project showcases robust features like conditional routing, efficient MongoDB updates using the $inc operator for inventory management, and a professional, responsive UI/UX using Tailwind CSS and Framer Motion.

Key Implementation Challenges

  • Atomic Inventory Management: Utilizing the MongoDB $inc operator to ensure reliable, concurrent updates (decrement on purchase, increment on cancellation) to product quantities, which is critical for a high-volume B2B platform.
  • Conditional Views and Filtering: Implementing dynamic state logic for the "All Products" page to allow users to toggle between Card and Table views and filter only products with high minimum stock (minimum_selling_quantity > 100).
  • Secure Transaction Flow: Ensuring that the checkout process is fully protected by JWT, and that product data remains consistent during transactions, particularly validating quantity against the minimum selling limit.
  • Form Handling: Managing complex, multi-field product submission forms using React Hook Form for validation and efficient data submission.

Potential Improvements & Future Plans

  • Dedicated Vendor Dashboard: Implement a separate dashboard for manufacturers/distributors to view sales reports, manage product performance metrics, and handle disputes, moving toward a true multi-vendor platform.
  • Multi-Currency Support: Integrate a currency exchange API to allow international buyers to view prices in their local currency, simplifying global trade and expanding the platform's reach.
  • Logistics & Tracking: Introduce API integrations with major global freight carriers to provide real-time shipping quotes and tracking information directly within the platform.
