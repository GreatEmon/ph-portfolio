Project Overview

RoomX is a dynamic Roommate Finder Website built on the MERN stack (MongoDB, Express, React, Node.js). It provides a secure and modern platform for users to connect based on crucial criteria like location, budget, and lifestyle preferences.

The application allows users to create detailed profiles, add their own room listings, and securely interact with interested users. Key features include protected routes, Google Sign-In integration, and interactive engagement tools like the "Like" system.

Key Implementation Challenges

  • Secure Contact Reveal: Implementing the logic to only reveal the contact number after a user clicks the "Like" button (showing genuine interest), improving security and reducing spam.
  • Protected Routes & User State: Managing user authentication and role-based state using Firebase Authentication on the client and coordinating access control via JWT on the server for private routes (e.g., `/add-roommate`).
  • Preference-Based Search: Designing the MongoDB schema to efficiently filter and search listings based on multiple, multi-valued lifestyle preferences (e.g., interests, cleaning habits, party tolerance).
  • Modern UX: Integrating complex third-party libraries like Lottie React for smooth, modern animations while maintaining fast load times and overall responsiveness.

Potential Improvements & Future Plans

  • Chat Integration: Implement a private, real-time chat feature (using WebSockets/Socket.io) to allow interested parties to communicate directly within the platform without revealing personal contact information initially.
  • Advanced Matching Algorithm: Develop a more sophisticated matching score that weighs compatibility factors (budget, location, and lifestyle) to suggest the top 5 compatible roommates for any given listing.
  • Image Upload Service: Integrate a dedicated service like Firebase Storage or Cloudinary to handle multiple image uploads for listings, instead of relying on simple URLs.
